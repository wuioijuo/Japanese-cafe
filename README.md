# Japanese — сайт японского кафе

Japanese — адаптивный многостраничный сайт для японского кафе.  
Проект выполнен как учебная проектная работа по HTML, CSS и JavaScript.

## Посмотреть сайт

[Открыть сайт Japanese](https://wuioijuo.github.io/Japanese-cafe/)

## Страницы сайта

- Главная страница
- Новости
- Меню
- Заказ
- Контакты

## Что реализовано

- единый дизайн сайта;
- адаптивная верстка под мобильные устройства;
- бургер-меню;
- страница новостей с модальными окнами;
- каталог блюд с карточками на flex-верстке;
- форма заказа с обработкой данных на сайте;
- сохранение заказа в localStorage;
- вывод текущей даты через JavaScript;
- слайдер рекомендаций;
- карта Яндекс и геолокация на странице контактов.

## Технологии

- HTML5
- CSS3
- JavaScript
- Яндекс.Карты
- LocalStorage
